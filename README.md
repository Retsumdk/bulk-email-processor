## Features

- Production-ready code
- TypeScript with full type safety
- Comprehensive error handling
- Built by Retsumdk

## Installation

```bash
git clone https://github.com/Retsumdk/bulk-email-processor.git
cd bulk-email-processor
bun install
```

## Usage

```bash
bun run src/index.ts --help
```
