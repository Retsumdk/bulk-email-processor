# bulk-email-processor

Queue-based bulk email processing with templates

## Features

- Production-ready code
- TypeScript/Python with full type safety
- Comprehensive error handling
- Built with Zo Computer by The BookMaster

## Installation

```bash
git clone https://github.com/Retsumdk/bulk-email-processor.git
cd bulk-email-processor
bun install
```

## Usage

```bash
bun run src/index.ts --help
```

## Configuration

Create `config.json` (TypeScript) or `config.yaml` (Python) for custom settings.
